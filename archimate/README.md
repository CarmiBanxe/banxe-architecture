## How to export from Archi

### Open Exchange XML (recommended вЂ” contains views)

1. Open the Banxe model in Archi
2. **File в†’ Export в†’ Model в†’ Open Exchange XML Format...**
3. Save as `archimate/banxe-model.xml`
4. Run `make import-archimate`

### CSV Export

1. Open the Banxe model in Archi
2. **File в†’ Export в†’ Model в†’ CSV...**
3. Select output folder: `archimate/csv/`
4. Archi creates: `elements.csv`, `relations.csv`, `properties.csv`
5. Run `make import-archimate`

## How to run the import pipeline

```bash
# From banxe-architecture root:
make import-archimate

# Or directly:
python3 scripts/import-archimate.py

# With validation against SERVICE-MAP.md:
python3 scripts/import-archimate.py --validate

# Custom paths:
python3 scripts/import-archimate.py \
  --xml archimate/banxe-model.xml \
  --csv-dir archimate/csv \
  --output-dir archimate/parsed
```
