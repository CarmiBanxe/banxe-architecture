# Spec Writing Skill

## Формат IL (Instruction Ledger):
IL-XXX | Статус | Приоритет | Описание | Блокировки | Дата

## Файлы для обновления:
- INSTRUCTION-LEDGER.md -- создание/обновление IL-записей
- COMPLIANCE-MATRIX.md -- обновление при закрытии задач
- BLOCKED-TASKS.md -- при блокировках
- PLANES.md -- стратегическое планирование

## Статусы:
OPEN -> IN_PROGRESS -> REVIEW -> DONE | BLOCKED

## Приоритеты:
P0 (критический) > P1 (важный) > P2 (обычный)
